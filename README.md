# 问道长生 - 仙侠放置游戏

一个大型仙侠题材的放置类游戏，玩家可以修炼境界、学习功法、炼制法宝、捕捉灵宠、建设宗门、挑战世界BOSS。

## 游戏特色

- 🎮 **境界系统** - 从练气到仙人，追求长生之道
- 📚 **功法系统** - 10+种功法，各具特色
- ⚔️ **法宝系统** - 炼制强化专属法宝
- 🐲 **灵宠系统** - 捕捉培养神兽伙伴
- 🏛️ **宗门系统** - 建设自己的仙门
- 👹 **世界BOSS** - 挑战强大BOSS获取奖励

## 在线试玩

访问 https://ziwensay.github.io/xianxia-game/ 开始游戏

## 本地运行

```bash
# 克隆项目
git clone https://github.com/ZiWenSay/xianxia-game.git

# 进入目录
cd xianxia-game

# 使用任意HTTP服务器
# Python
python -m http.server 8000

# 或 Node.js
npx http-server

# 然后访问 http://localhost:8000
```

## 游戏玩法

1. **修炼** - 点击修炼获得灵气，提升境界
2. **灵根** - 升级灵根提高灵气获取效率
3. **功法** - 解锁学习各种功法提升战力
4. **法宝** - 炼制装备强力法宝
5. **灵宠** - 捕捉灵兽助战
6. **宗门** - 建设建筑提升属性
7. **BOSS** - 挑战世界BOSS获取大量奖励
